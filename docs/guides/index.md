## Prerequisites at a Glance

- Elixir ~> 1.15, Erlang/OTP 26+
- Node.js 20+ (asset pipeline)
- PostgreSQL 14+
- Python 3.10+ with `demucs`, `librosa`, `spotdl`
- Docker (for production deployment)
